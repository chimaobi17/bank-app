# ApexBank

A premium, private treasury banking application built with Laravel 12, React (Inertia.js), and Oracle Database 21c.

## Features
- **Executive Dashboard**: Real-time intelligence and portfolio value tracking.
- **Secure Authentication**: High-end Sign In and Sign Up portals.
- **Account Management**: Support for multiple account types with detailed history.
- **Oracle Powered**: Robust, enterprise-grade data management.

## Setup
1. Clone the repository
2. Run `composer install`
3. Run `npm install`
4. Configure your `.env` with Oracle credentials
5. Run `php artisan migrate`
6. Run `npm run build`
7. Start the server: `php artisan serve`
